&和&&、|和||的区别

Java Java 2017/01/21
  1. &和&&的区别：
  • 单&时，左边无论真假，右边都进行运算；
  • 双&时，如果左边为真，右边参与运算，如果左边为假，那么右边不参与运算。
  1. “|”和“||”的区别:
  • 单或时，左边无论真假，右边都进行运算；
  • 双或时，左边为真，右边不参与运算。

注：这里的左边右边值逻辑运算符的左边右边

异或( ^ )与或( | )的唯一不同之处是：当左右都为true时，结果为false。
