JS变量命名规范

前端 javascript 2017/01/10

1、必须以字母、下划线或美元符号开头，后面可以跟字母、下划线、美元符号和数字。如下:

正确: 

hello            
_hello         
$hello

错误:

6hello      //开头不能用数字
%hello     //开头不能用除(_ $)外特殊符号,如(%  + /等)
hello+js //开头中间不能使用除(_ $)外特殊符号，如(%  + /等)

2、变量中区分大小写，如A与a是两个不同的变量名。

3、不允许使用JavaScript关键了和保留字作变量名。
